# Data Friction Research

## Research Objective
Analyzing the trade-offs between Vertical Data Integration and Horizontal Market Frictions using the Three-Gauge Framework (Privacy, UX, and Cost).

## Structure
- **01_Source/**: Scripts and automation for data processing.
- **02_Raw/**: Immutable data sources (TCF JSON, cookie logs).
- **03_Analysis/**: Python logic, notebooks, and thesis results.
